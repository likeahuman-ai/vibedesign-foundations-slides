# Vibedesign Foundations — Workshop Slides

A 14-slide HTML deck for the **Brand Design Foundations with AI** workshop at Norrsken House Barcelona, May 12, 2026.

Co-led by Jasper Staats (Like a Human) and Lilibeth Bustos.

## Open

```bash
open vibedesign-foundations.html
```

Self-contained — no build step, no dependencies. All fonts via Google Fonts CDN. All other assets in `assets/`.

## Keyboard

| Key | Action |
|---|---|
| `↑ ↓ ← →` or `Space` | Navigate slides |
| `F` button (top-right) | Toggle fullscreen |
| `A` / `B` on slide 5 | Toggle background concept (Krea loop / LAH illustration) |

## Editorial system

Two systems coexist in the deck right now:

- **LAH brand mockup system** — Montserrat, DLS palette (navy / cream / lime), tonal gradients, atmospheric illustrations from the LAH brand library. Used on most slides.
- **Lilibeth DNA system** — extracted from the *Your Moodboard* deck. Off-white paper, near-black ink, single acid-lime accent, hairline rules, light-weight display type, generous negative space. Applied to slide 7 (Colors) as the proof-of-concept. Reference: `visual-dna-lilibeth.md`.
